## Resumen de lo aprendido

### Lección 1 — Variables y Tipos de Datos
- `let` para constantes, `var` para variables
- Tipos: `String`, `Int`, `Double`, `Bool`
- Interpolación de strings: `"\(variable)"`

### Lección 2 — Condicionales
- `if`, `else if`, `else`
- Operadores: `==`, `!=`, `>`, `<`, `>=`, `<=`, `&&`

### Lección 3 — Loops
- `for i in 1...10` — rangos con `...` y `..<`
- Operador módulo `%` para saber si un número es par
- Combinar `for` con `if` para filtrar

### Lección 4 — Funciones
- `func nombre(parametro: Tipo) -> TipoRetorno`
- Etiquetas de parámetros al llamar: `saludar(nombre: "Nicolas", edad: 25)`
- Funciones con y sin valor de retorno

### Lección 5 — Arrays y Diccionarios
- Array: lista ordenada, acceso por índice `notas[0]`
- Diccionario: pares clave-valor `[String: String]`, acceso por clave
- `.count`, `.append()`

### Lección 6 — Optionals
- `String?` — variable que puede ser `nil`
- `if let` — optional binding, forma segura de desenvolver
- `??` — nil coalescing, valor por defecto si es nil
- `!` — force unwrap, peligroso, evitar en producción
